# ToolFlow ランディングページ

ToolFlowは、毎日の作業やタスクをシンプルに整理するためのサービスを紹介する、1ページ構成のランディングページです。

ランディングページとは、サービスの魅力を分かりやすく伝え、利用を始めてもらうためのWebページのことです。

## このプロジェクトでできること

- ToolFlowがどんなサービスなのかを紹介する
- タスクの整理、進捗の確認、続けやすさという3つの特長を伝える
- 「無料で始める」ボタンから、利用開始の案内へ進める
- パソコンだけでなく、スマートフォンでも読みやすく表示する

## 使用している技術

### HTML

ページに表示する文章、見出し、ボタンなどの**骨組み**を作るために使っています。

### CSS

文字の大きさ、色、余白、レイアウトなどの**見た目**を整えるために使っています。画面の幅に応じてレイアウトを変えることで、スマートフォンにも対応しています。

## ファイル構成

- `index.html`：ページの内容と構造
- `style.css`：ページのデザインとレイアウト

## 表示方法

`index.html` をWebブラウザで開くと、ランディングページを確認できます。
